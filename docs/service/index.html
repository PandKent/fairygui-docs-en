社区版和专业版编辑器

FairyGUI编辑器有两个版本，社区版和专业版。社区版完全免费，并且可以用于商业项目，没有任何限制；专业版采用订阅收费模式，功能更加强大，推荐您购买专业版。

  • 微信支付。请扫码关注公众号，然后选择专业版。注册码将通过公众号下发，请务必关注。

  • PayPal支付
